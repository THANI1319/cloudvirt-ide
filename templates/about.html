About CloudVirt Nexus

Batch No: 20 | Department of Information Technology

Project Overview

CloudVirt Nexus is an advanced simulation and analytics platform built to demonstrate the performance differences between Hardware-level Virtualization (Virtual Machines) and OS-level Virtualization (Docker Containers).

Our primary objective is to help cloud developers and students dynamically analyze workload execution times, memory footprints, and CPU overheads before deploying applications to real-world cloud environments.

Simulation Accuracy Engine

How accurate are our metrics? The platform uses Python's psutil library to capture Real-time System Metrics during the IDE code execution, combined with scientifically proven baseline constants.

  • Live Execution Baseline: Captures exact CPU cycles and RAM used by the pure Python process.
  • VM Emulation: Injects +2.8s Boot Time, +120MB RAM, and +18% CPU to simulate Hypervisor & Guest OS weight.
  • Docker Emulation: Injects minimal overhead (+0.1s Boot, +12MB RAM) to reflect kernel sharing efficiency.
